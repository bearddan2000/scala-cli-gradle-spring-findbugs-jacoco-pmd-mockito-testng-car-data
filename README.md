# scala-cli-gradle-spring-findbugs-jacoco-pmd-mockito-testng-car-data

## Description
A POC for spring app using testng, findbugs,
jacoco, pmd, mockito framework.

## Tech stack
- scala
- gradle
  - spring
  - testng  
  - mockito
  - jacoco
  - pmd
  - findbugs

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
